# Flowglad Pay Skills

AI agent skills for working with [Flowglad Pay](https://app.flowgladpay.com) payment cards.

## Installation

```bash
npx skills add flowglad/pay-skills
```

## Available Skills

| Skill | Description |
|-------|-------------|
| `card-sessions` | Create and redeem card sessions to securely access card details (PAN, CVV, expiry) |
| `agent-card-setup` | Onboard an agent with spending policies and a virtual card |

## Supported Agents

- Claude Code
- Cursor
- Codex
- Windsurf
- Other skills.sh compatible agents

## Documentation

For full Flowglad Pay documentation, visit [app.flowgladpay.com](https://app.flowgladpay.com).
